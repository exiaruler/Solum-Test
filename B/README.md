# A. CargoCraft Fleet


## Prerequisites

Before running the project, make sure you have:

- Java Development Kit (JDK) 17 or later installed
- An IDE:
  - Visual Studio Code (with Java extensions installed)

## Running the Code in Visual Studio Code

1. Open the Main.java file in A folder
2. Locate your main class (the file containing `public static void main(String[] args)`).
3. At the top of the editor, VS Code will show a "Run" or "Run button.
4. Click Run to execute the program.
5. Output will appear in the Terminal or Debug Console.
